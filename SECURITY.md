# Security Policy

## Supported Versions

This repository is provided primarily for evaluation/demo purposes. Security fixes are applied on a best-effort basis.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please **do not open a public issue**.

Instead, report it privately:

- Telegram: @ogsulem

Please include:

- A clear description of the issue and impact
- Steps to reproduce
- Affected endpoints / components
- Any proof-of-concept information

We will acknowledge receipt and follow up with next steps.
